examples_progress_bar module¶

examples_progress_bar.ex1()¶

Example 1

pb1 = ProgressBar(100, '%', pre='Downloading file', post='Download finished', length=25)

pb1.print_progress()  # Prints the initial empty progress bar

for mb in range(1, 101):

pb1.print_progress(mb)

sleep(0.15)

Output:

Downloading file

[========================>] - Finished 100 % of 100 %

Download finished

examples_progress_bar.ex3()¶

Example 3

pb3 = ProgressBar(1000.12, ‘MB’, pre=’Downloading file’, post=’Download finished’, length=100) pb3.print_progress() # Prints the initial empty progress bar for mb in range(1, 1001):

if mb != 1000 and mb % 2 == 0:

mb = mb + 0.5

elif mb != 1000:

mb = mb + 0.25

else:

mb = mb + 0.12

pb3.print_progress(mb)

sleep(0.025)

Output:

Downloading file [===================================================================================================>] - Finished 1000.12 MB of 1000.12 MB

Download finished
